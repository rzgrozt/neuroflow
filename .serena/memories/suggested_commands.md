# Suggested Commands for NeuroFlow Development

## Running the Application
```bash
python neuroflow.py
```

## Installing Dependencies
```bash
pip install -r requirements.txt
```

## Code Formatting (Black)
```bash
black neuroflow.py
```

## Syntax Check
```bash
python -m py_compile neuroflow.py
```

## Git Operations
```bash
git status
git add neuroflow.py
git commit -m "message"
git push
```

## System Commands (Linux)
```bash
ls -la          # List files
cat filename    # View file
grep pattern .  # Search
find . -name "*.py"  # Find files
```
